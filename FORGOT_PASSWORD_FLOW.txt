║                            SECURITY FEATURES                                  ║
╚═══════════════════════════════════════════════════════════════════════════════╝

✅ Username + Email Validation (prevents email-only attacks)
✅ No Information Leakage (generic error messages)
✅ Time-Limited OTP (10-minute expiration)
✅ Single-Use OTP (deleted after successful reset)
✅ OTP Replacement (new request invalidates old OTP)
✅ Password Hashing (bcrypt with salt)
✅ Input Validation (trim whitespace, format checks)
✅ Audit Logging (console logs for monitoring)

╔═══════════════════════════════════════════════════════════════════════════════╗
║                            ERROR SCENARIOS                                    ║
╚═══════════════════════════════════════════════════════════════════════════════╝

❌ Invalid Username          → "Invalid Username or Gmail ID"
❌ Wrong Email for Username  → "Invalid Username or Gmail ID"
❌ OTP Not Found             → "OTP not found or expired"
❌ OTP Expired               → "OTP has expired. Please request a new OTP."
❌ Wrong OTP                 → "Invalid OTP"
❌ Passwords Don't Match     → "Passwords do not match"
❌ Password Too Short        → "Password must be at least 6 characters long"
